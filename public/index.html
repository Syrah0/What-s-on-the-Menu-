      Sobre el proyecto

      Proyecto final del ramo CC3201 - Bases de Datos, trabajo realizado en grupo. Se pedía instalar una base de datos en un servidor dado, con cierta cantidad de tablas que vieran un tema en especifico. Se pedía hacer optimzación a estas tablas y crear vistas útiles. Finalmente había hacer una aplicación web que fuera capaz de hacer las consultas a esta base de datos.
